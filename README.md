# 🚀 Meu Portfólio Full Stack

Bem-vindo ao repositório do meu portfólio pessoal! Este é um sistema Full Stack (Monorepo) desenvolvido para demonstrar minhas habilidades em construir aplicações completas, desde a arquitetura do banco de dados até a interface de usuário.

## 💻 Tecnologias Utilizadas

**Frontend:**
* [Next.js](https://nextjs.org/) (React Framework)
* TypeScript
* Tailwind CSS & Shadcn/UI

**Backend:**
* [Java 17+](https://www.java.com/)
* [Spring Boot](https://spring.io/projects/spring-boot) (REST API)
* PostgreSQL (Conteinerizado)

**Infraestrutura:**
* Docker & Docker Compose

---

## ⚙️ Como rodar o projeto localmente

### Pré-requisitos
Certifique-se de ter instalado em sua máquina:
* Git
* Docker Desktop
* Node.js (v18+)
* Java JDK 17+ e Maven
